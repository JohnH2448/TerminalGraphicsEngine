# PyFrame
## Dead Simple Terminal Driven Graphics Engine
### Description
PyFrame is a graphics engine that renders live frames to a terminal. The engine is build for beginners and experts alike to use with minimal implimentation. Everything needed to build a game, demo, or even livesteam is included in a single script designed for maximum portability and ease of use.
